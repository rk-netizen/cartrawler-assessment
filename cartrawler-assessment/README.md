# CarTrawler Assessment

This is a simple car rental UI built with React and Vite.

## How to run it

1. Install everything you need:
    ```
    npm install
    ```
2. Start the app:
    ```
    npm run dev
    ```
3. Open your browser and go to [http://localhost:5173](http://localhost:5173)
